# Graalvm build image

To build graalvm native application with maven installed.

## Dockerfile example with quarkus application

```
FROM shyndard/graalvm-native-build:java11-maven363
WORKDIR /build/
COPY . /build/
RUN mvn -f /build/pom.xml package -Pnative
```
